
🪔 Diwali Sales Analytics & Customer Insights Analysis
📌 Project Overview
This project focuses on analyzing Diwali sales data to understand customer purchasing behavior, demographic trends, product demand, and sales performance using data analytics techniques. The repository publicly includes a Jupyter Notebook file named DataAnalysis.ipynb and a dataset file named Diwali Sales Data.csv, indicating a notebook-based analysis workflow built around Diwali sales data. 

The analysis is designed to transform raw transactional data into meaningful business insights that can support marketing strategy, customer targeting, product planning, and festival-season sales optimization. A notebook-based workflow is commonly used for loading data, cleaning records, performing exploratory analysis, and presenting visual findings step by step. 

🎯 Problem Statement
The objective of this project is to analyze Diwali sales data and answer important business questions such as: 

Which customer segments contribute the most to total sales? 

Which product categories perform best during the Diwali season? 

How do demographic factors such as gender, age group, marital status, and occupation influence purchasing behavior? 

Which states or regions generate the highest order volume and revenue? 

What insights can businesses use to improve festive campaign performance and customer engagement?

🔍 Key Business Insights
Typical Diwali sales analyses show that festive purchase behavior can be segmented effectively using demographic and category-level trends, helping businesses identify their strongest customer groups and best-performing product lines. 

Common project findings in this type of dataset include stronger contribution from specific age groups, category-level concentration in top-selling products, and measurable differences in spending behavior across customer occupations, regions, and marital status groups.
